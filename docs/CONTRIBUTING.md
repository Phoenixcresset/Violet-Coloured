# Requirements

- Git Bash

# Initializing the project

Run this script to be able to run shell files via terminal :

```shell
pnpm config set scriptShell "C:\\Program Files\\git\\bin\\bash.exe"
```

BEFORE cloning, if you are on Windows, you must :

- Enable [Developper Mode](https://learn.microsoft.com/en-us/windows/advanced-settings/developer-mode)
- Execute this script :

  ```shell
    git config --global core.symlinks true
  ```
